# singular-plural-json [![NPM version](https://badge.fury.io/js/singular-plural-json.svg)](https://badge.fury.io/js/singular-plural-json)

## Installation

```
npm i --save-dev singular-plural-json
```

## Usage

```javascript
var spJson = require('singular-plural-json')
console.log(spJson)
// {book:"books",dog:"dogs", ... }
```
